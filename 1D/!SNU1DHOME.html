Welcome to SNU!

You are using 1 dimensional mode. This mode is text only. It is good for older browsers in a virtual machine, or a slow internet connection, or if you just want to focus on text

Compatibility

SNU 1D mode is made to be compatible with older web browsers and slow internet connections, so by default, UTF-16 is turned off. In fact, all modern encoding is turned off. This page is ASCII

Distraction free

1D mode has a lot less going on and it makes a less distracting environment. There are no videos to watch, no images to look at, no sound to listen to, it is all just text

Note: this mode is not compatible with blind users, as the voice search features are disabled. If you are letting a blind person use SNU, please try 2D or 3D mode
